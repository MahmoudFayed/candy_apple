当面は日本語のみです。

## プロジェクト名の由来

やおきん製のダイヤモンドリングキャンディ、または縁日で見かけるりんご飴からです。

最初、エジプトやサウジアラビアのお菓子を探して、そこから採る予定でしたがしっくり来なかったものでして。

続きは後で書きます。


## 記事の寄稿について
